# Installation

scikit-opls requires Python 3.13+.

```bash
pip install scikit-opls
```

Plotting ([`scikit_opls.plotting`](api/plotting.md)) needs `matplotlib`, which is
an optional extra:

```bash
pip install "scikit-opls[plot]"
```

## From source (with uv)

```bash
git clone https://github.com/HauserGroup/scikit-opls
cd scikit-opls
uv sync --dev
```
